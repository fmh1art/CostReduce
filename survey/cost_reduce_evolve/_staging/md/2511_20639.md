end training-free framework that enables pure la-
tent collaboration among LLM agents. In Latent-
MAS, each agent first performs auto-regressive
latent thoughts generation through last-layer hid-
den embeddings instead of text. Then, a shared

1. Introduction
Model collaboration emerges as the foundation of system-
level intelligence in the era of Agentic AI (Acharya et al.,
2025). Recent advances in multi-agent systems (MAS) (Wu
et al., 2024; Hu et al., 2025) have catalyzed a paradigm
shift from solitary, model-centric reasoning into a collabora-

tive endeavor among multiple interacting models. Among
these, large language model (LLM)-based MAS has been
adopted across various downstream applications, including
cooperative math and science reasoning (Pezeshkpour et al.,
2024; Zhou et al., 2025), automated code generation (Yang
et al., 2024b; Ding et al., 2025), and open-domain question
answering (Jin et al., 2025; Li et al., 2025c; 2026). Within
LLM-based MAS, natural language or text generally serves
as the lingua franca – the common medium that carries each
agent’s internal thoughts and enables communication across
different agents (Guo et al., 2024).
Beyond explicit text, several studies have explored the use
of LLMs’ continuous latent space as a new form of “model
language,” (Chen et al., 2025b) by either (i) leveraging hid-
den representations within transformers to enable single
model’s internal latent chain-of-thought (CoT) reasoning
(Hao et al., 2024; Zheng et al., 2025; Zhang et al., 2025),
or (ii) employing KV caches or layer embeddings for in-
formation exchange across two models (Liu et al., 2024;
Fu et al., 2025). However, a comprehensive model col-
laboration framework unifying both latent reasoning and
latent communication remains unexplored. Moving one step
forward, we aim to investigate:
Can MAS achieve pure latent collaboration?
To address this question, we introduce LatentMAS, an end-
to-end collaborative framework that operates entirely within
the continuous latent space. Our core design integrates both
internal latent thoughts generation and cross-agent latent
working memory transfer. Inside each agent, reasoning
unfolds through auto-regressive generation of last-layer hid-
den representations, capturing the model’s ongoing internal
thoughts without explicit decoding. Across agents, informa-
tion is exchanged via shared latent working memory stored
in layer-wise KV caches, capturing both the input context
and newly generated latent thoughts. Overall, LatentMAS
is completely training-free, enabling all agents to think and
interact purely through their internal latent representations.
Building on this framework design, LatentMAS is grounded
on three foundational principles, which are substantiated
through comprehensive theoretical and empirical analyses:
• Reasoning Expressiveness: Hidden representations
naturally encode model continuous thoughts, making
each latent step more expressive than discrete tokens
and capable of conveying richer semantic information.
• Communication Fidelity: Latent working memory
preserves input representations and latent thoughts,
enabling lossless information transfer across agents.
• Collaboration Complexity:
LatentMAS exhibits
higher collaborative expressiveness than text-based
MAS with substantially lower inference complexity.
The first two principles jointly underscore the advantage of
LatentMAS by enabling richer latent reasoning and lossless
latent communication. The last principle characterizes how
latent collaboration reshapes the efficiency-expressiveness
tradeoff, illustrating that LatentMAS enables scalable multi-
agent coordination with reduced computational overhead.
To empirically assess the efficacy of LatentMAS, we con-
duct comprehensive evaluations on 9 benchmarks spanning
math and science reasoning, commonsense understanding,
and code generation (Figure 1). Across both sequential
and hierarchical MAS settings, and over five different back-
bone models spanning the Qwen3 (Yang et al., 2025) and
Llama3 (Grattafiori et al., 2024) families, LatentMAS con-
sistently outperforms advanced single-model and text-based
MAS baselines by (i) improving accuracy by up to 14.6%,
(ii) reducing output token usage by 70.8%-83.7%, and (iii)
delivering 4×-4.3× faster end-to-end inference. These re-
sults demonstrate that latent collaboration not only enhances
system-level reasoning quality but also provides consistent
efficiency gains without any additional training. Further
detailed analyses of latent thought expressiveness, working-
memory transfer, and input-output alignment confirm that
LatentMAS enables semantically meaningful, lossless, and
stable collaboration entirely in latent space.
2. Preliminary and Notations
Auto-regressive Generation in Transformer. Let fθ(·)
denotes the function computed by a standard Transformer
model (Vaswani et al., 2017), parameterized by θ. Given
an input sequence x = (x1, x2, . . . , xT ), the transformer
fθ(·) first encodes each token via its input embedding
layer Win to obtain token embeddings up to step t, i.e.,
E = [e1, e2, . . . , et] ∈Rt×dh, where dh is the model’s hid-
den dimension. The input token embeddings E then succes-
sively process through L transformer layers in the forward
pass through the model’s residual stream, yielding the final-
layer hidden representations H = [h1, h2, . . . , ht] ∈Rt×dh.
For next token generation, the model computes:
fθ(xt+1 | x≤t) = softmax(htWout),
(1)
where Wout denotes the language model head that maps the
hidden representation to the vocabulary space. Each token
is generated in an auto-regressive manner and appended to
the input sequence. For latent generation, the model per-
forms auto-regression over hidden representations instead
of tokens, bypassing explicit decoding (Hao et al., 2024).

working memory during auto-regressive generation, stor-

steps to avoid redundant computation. Specifically, given
the input embeddings E, each transformer layer projects
them through projection matrices WQ, WK, WV to obtain

3. LatentMAS
We introduce LatentMAS, an end-to-end latent collabora-
tion framework that all agents reason and communicate
entirely within the latent space. As illustrated in Figure 3,
our method enables LLM agents to (i) auto-regressively gen-
erate expressive latent thoughts (Section 3.1), (ii) transfer
latent working memory across agents with lossless com-
munication fidelity (Section 3.2), and (iii) achieve lower
inference complexity than text-based MAS while support-
ing a higher degree of model coordination (Section 3.3).
Method Roadmap.
In the following sub-sections, we
present the complete pipeline of LatentMAS, detailing each
component and interleaving theoretical analyses to justify
the corresponding design principles.
3.1. Auto-regressive Latent Thoughts Generation
We first describe how each LLM agent performs latent rea-
soning through its layer-wise hidden states. Instead of de-
coding explicit tokens, reasoning unfolds directly within the
agent by auto-regressively appending hidden representations
from the final model layer.
Given the input embeddings E = [e1, e2, . . . , et] containing
the information from the question q and each agent’s instruc-
tion prompt, each LLM agent Ai ∈A passes E through
L transformer layers to compute the last-layer hidden rep-
resentation ht at current step t. Then, we insert ht as the
input embedding for the next step t + 1, replacing the origi-
nal decoding and next-token embedding processes used in
standard token generation. We auto-regressively repeat the
process for m latent steps, yielding a sequence of newly gen-
erated last-layer hidden states H = [ht+1, ht+2, . . . , ht+m].
We define the continuous output representations H as the
latent thoughts generated by Ai.
Input-Output Distribution Alignment. Since the newly
generated H form a sequence of dense, high-level repre-
sentations, directly inserting them into shallow layers as
input embeddings may lead to out-of-distribution activa-
tions (Meegahapola et al., 2019; Zhou et al., 2019) , as these
hidden states differ from the statistical patterns of learned to-
ken embeddings. To mitigate this in a training-free manner,
we propose a linear alignment operator that maps last-layer
hidden states back to the valid input embeddings. Specifi-
cally, given Win, Wout as the input and output embedding
layers of Ai, we seek a projection matrix Wa ∈Rdh×dh
that maps each output vector h ∈H to a new input vector e
to align with valid input space defined by Win:

working memory transfer guarantees information fidelity
equivalent to explicit input exchange.

working memory from preceding agents are equivalent
to those obtained when directly inputting the preceding
agents’ outputs.

avoid redundant recomputation for the successive agent.

4. Empirical Evaluations
Tasks and Datasets. We conduct comprehensive evalua-
tions of LatentMAS across 9 benchmarks spanning both
general-purpose and reasoning-intensive tasks: (i) Math &
Science Reasoning, including GSM8K (Cobbe et al., 2021),
AIME24 (Maxwell-Jia, 2024), AIME25 (math ai, 2025),
GPQA-Diamond (Rein et al., 2023), and MedQA (Yang
et al., 2024a); (ii) Commonsense Reasoning, including ARC-
Easy (Clark et al., 2018b) and ARC-Challenge (Clark et al.,
2018a); and (iii) Code Generation, including MBPP-Plus
(Liu et al., 2023) and HumanEval-Plus (Liu et al., 2023).
Additional benchmark details are provided in Appendix C.1.
Models and Baselines. We adopt 5 off-the-shelf models
from both the Qwen3 family (Yang et al., 2025) (4B, 8B,
and 14B) and the Llama 3 family (Grattafiori et al., 2024)
(3B, 8B) to construct LatentMAS at different scales. For
baseline comparison, we evaluate LatentMAS against: (i)
Single LLM agents (Single), where a single LLM directly
performs standard auto-regressive generation with token-
level decoding; (ii) Sequential text-based MAS (Sequen-
tial TextMAS), following the chain-of-agents design (Zhang
et al., 2024b) with text-mediated reasoning and communi-
cation; and (iii) Hierarchical text-based MAS (Hierarchi-
cal TextMAS), where domain-specialized agents collaborate
through a summarizer (Zhuge et al., 2024) using text-based
reasoning and communication. Detailed model and baseline
implementations are provided in Appendix C.2.

4.2. Efficiency Analyses on Latent Collaboration.
End-to-end Inference Speed. As established earlier in
Theorem 3.1, LatentMAS can achieve orders-of-magnitude
higher expressiveness, thereby enabling more efficient infer-
ence than text-based MAS. We further empirically validate
this advantage through comparisons with TextMAS. As
shown in Figure 4 and 10 (left), even after accelerating the
TextMAS baselines using the vLLM service, LatentMAS
still achieves a 2.6×-7× speedup over the vLLM-optimized
TextMAS. This improvement stems from the substantially
reduced number of latent steps required for latent thoughts
generation compared with the much larger decoding steps
needed for per-token text generation. With fewer than 50
latent steps, LatentMAS attains comparable or even higher
performance on reasoning-intensive tasks such as AIME
24/25, whereas TextMAS typically requires more than 20K
output tokens to complete full text-based CoT trajectories.
Token Throughput. In addition, as illustrated in Figure 4
and 10 (right), LatentMAS reduces token usage by 59.4%-
87.9% compared with TextMAS, as agents in LatentMAS
communicate by directly transferring latent working mem-
ory into another agent’s internal layers rather than relying
on the text medium. LatentMAS also achieves 15.0%-60.3%
lower token usage than single agents. Compared with single-
model reasoning, LatentMAS distributes the input question
across collaborating agents, greatly reducing the burden on
the final agent, which primarily aggregates preceding latent
thoughts and decodes the final answer using only a small
number of tokens. As a result, the entire system generates
fewer output tokens while still achieving higher accuracy.

6. Related Work
LLM-based Multi-agent Systems. Recent studies in Agen-
tic AI have extended classical multi-agent systems (Hong
et al., 2023; Park et al., 2023a; Yang et al., 2024c) grounded
in traditional reinforcement learning and policy coordi-
nation, to modern LLM settings (Li et al., 2025b; Tan
et al., 2025), enabling models to operate as autonomous
agents that collaborate in reasoning, planning, and problem-
solving (Tao et al., 2024; Wang et al., 2025c; Zhao et al.,
2025). Early methods (Yao et al., 2022; Wu et al., 2024; Li
et al., 2023) coordinate multiple LLMs through explicit dia-
logue and role assignment. Additional methods introduce
structured communication protocols to enhance cooperation
efficiency (Ye et al., 2025a) and emergent specialization
(Mieczkowski et al., 2025; Huang et al., 2026) among agents.
Building upon these advanced text-MAS methods, our work
aims to construct a multi-agent collaboration system entirely
in latent space, treating agents as tightly integrated compo-
nents to achieve more expressive and efficient coordination.
Model Collaboration in Latent Space. Recent studies
on model collaboration (Tran et al., 2025) have gradually
shifted from text-level coordination toward interaction di-
rectly in latent representations. Instead of exchanging infor-
mation through texts, these works (Coda-Forno et al., 2025;
Ye et al., 2025a; Fu et al., 2025; Zheng et al., 2025) enable
models to communicate through model internal states such
as hidden representations (Coda-Forno et al., 2025) for ef-
ficiency (Kong et al., 2026). Most existing methods rely
on auxiliary alignment mechanisms and typically require
additional training to facilitate effective cross-model com-
munication. On the other hand, LatentMAS is a training-free
latent multi-agent system in which each agent reasons and
communicates via native latent representations.
We leave additional related works on latent reasoning and
agent collaboration in Appendix H.
7. Conclusion
We introduce LatentMAS, a training-free framework that
enables multi-agent systems to collaborate entirely within
latent space. Our framework enables latent auto-regressive
reasoning with a lossless latent working-memory transfer
mechanism. Theoretical analyses and empirical evaluations
across diverse reasoning and generation benchmarks demon-
strate that LatentMAS consistently improves accuracy per-
formance, token usage, and decoding speed over strong
single-model and text-based MAS baselines. LatentMAS
establishes a scalable and general paradigm for agentic col-
laboration beyond the constraints of natural language.

Theorem B.2 (Restate of Theorem 3.3). In both latent and text-based reasoning, the outputs of an agent when receiving
latent working memory from preceding agents are equivalent to those obtained when directly inputting the preceding
agents’ outputs.
Proof. Let h(l), K(l), V (l) and h′(l), K′(l), V ′(l) denote the output, keys, and values of l-th transformer layer when receiving
latent working memory from preceding agents and when directly inputting the preceding agents’ outputs, respectively. In
the following, we will use induction to show that h(l) = h′(l) for every layer l = 1, . . . , L.
Induction step.
Suppose that h(l−1) = h′(l−1), and we will show that h(l) = h′(l).
The KV cache contains K≤t+m(l) and V≤t+m(l). For each past token layers, at each attention layer, the transformer produces
one column of K≤t+m(l) and a corresponding column of V≤t+m(l). At the next step the model forms a query from the
current input and then uses that query together with the stored K≤t+m(l) and V≤t+m(l) to form the attention result. That
attention result is a deterministic function of the query and of the keys and values it attends to.
We are comparing two ways to make those same keys and values available to the current computation: (i) actually feeding
the earlier tokens into the model again, in which case the model will recompute the same keys and values and then use them
in attention; (ii) reading in K≤t+m(l) and V≤t+m(l) from the cache and use them directly. In both cases the keys and values
presented to the attention computation are identical, because the cache was produced by the same model on the same inputs.
